# 👷‍♂️ 4. عرض الموظف

### US-13: عرض جدول الموظف

بصفتي **موظف**
أريد رؤية جدول عملي
لكي أعرف متى أعمل

### US-14: عرض تفاصيل الدوام

بصفتي **موظف**
أريد رؤية وقت كل شفت وتاريخه
لكي أكون جاهز للعمل

# ⏱️ 5. الحضور والانصراف

### US-15: تسجيل الحضور

بصفتي **نظام**
أريد تسجيل حضور الموظف (check-in / check-out)
لكي يتم حساب ساعات العمل

### US-16: تسجيل الغياب

بصفتي **مدير**
أريد تسجيل موظف كغائب
لكي يتم احتساب راتبه بشكل صحيح

# 💰 6. الرواتب (Payroll)

### US-17: حساب ساعات العمل

بصفتي **نظام**
أريد حساب عدد ساعات كل موظف من الشفتات والحضور
لكي أستخدمها في الرواتب 

### US-20: تصدير Excel

بصفتي **مدير**
أريد تصدير الرواتب إلى ملف Excel
لكي أستخدمه في المحاسبة 

### US-24: دعم قنوات متعددة

بصفتي **نظام**
أريد إرسال الإشعارات عبر Email وWhatsApp
لكي أصل لكل الموظفين

# 📱 8. تطبيق الموظف

### US-25: تسجيل الدخول في تطبيق الموظف

بصفتي **موظف**
أريد تسجيل الدخول للتطبيق
لكي أرى بيانات عملي

### US-26: عرض ساعات العمل

بصفتي **موظف**
أريد رؤية عدد ساعات عملي
لكي أعرف جهدي

### US-27: عرض الراتب الحالي

بصفتي **موظف**
أريد رؤية راتبي الحالي
لكي أعرف مستحقاتي

# ⚙️ 9. النظام العام

### US-28: إعداد النظام

بصفتي **مدير**
أريد إعداد معلومات الشركة
لكي تظهر في النظام

### US-29: تشغيل النظام على سحابة

بصفتي **مالك النظام**
أريد نشر النظام على سيرفر
لكي يكون متاح دائمًا
